# Sta je ovo?

Mala sajt uradjen U C# MVC da prati istoriju racunja bruto iznosa

# Potrebno
 - MSSQL
 - Net Core 3.1

# Konfigurisati
- Idite u /bruto-web I videcete `appsettings.json` Promenite ConnectionString za mssql I mozete pokrenuti
